Functions Included and their work:
1. void mainMenu(): Display the option for Admin or User login or exit.

2. void adminMenu(): Will ask for admin password first then will be able to view menu for admin.

3. void userMenu(): Display menu for user. (New or Old registration).

4.void close(): Will display the end of the Program.

5.void fordelay(int): Will producde some kind of delay in programs.

6.int authenticate(int, char []): Will authenticate user or admin's password (1 for admin 2 for user)

7.void encrypt(char pass[]): Encrypt the Password.

8.void decrypt(char pass[]): Decrypt the Password.
